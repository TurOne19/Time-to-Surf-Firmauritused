# Time to Surf — лендинг

Готовый лендинг на Next.js 14 + TypeScript + Tailwind CSS. Три языка: **эстонский** (по умолчанию), **английский**, **русский** — переключатель в шапке сайта.

## Как запустить

1. Установи Node.js версии 18.18 или новее (лучше 20+).
2. В папке проекта выполни:

```bash
npm install
npm run dev
```

3. Открой http://localhost:3000

## Как собрать для продакшена

```bash
npm run build
npm run start
```

Готовый сайт можно задеплоить на Vercel, Netlify или любой хостинг с поддержкой Node.js (просто загрузи всю папку и подключи к своему аккаунту на выбранной платформе).

## Как добавить свои фото и видео

Сейчас в галерее 63 места под фото и 4 места под видео — они уже показывают красивые песочные плейсхолдеры с номером, пока настоящих файлов нет.

Чтобы заменить плейсхолдеры на настоящие фото/видео — просто положи файлы в папку `public/gallery` с такими именами:

- Фото: `photo-01.jpg`, `photo-02.jpg`, … `photo-63.jpg`
- Видео: `video-01.mp4`, `video-02.mp4`, `video-03.mp4`, `video-04.mp4`
- Превью (постер) для видео: `video-01-poster.jpg` и т.д.

Как только файл появится в папке с правильным именем — плейсхолдер автоматически заменится на настоящее фото/видео, ничего в коде менять не нужно.

## Как поменять фото на Hero (первый экран)

Замени файл `public/hero.jpg` на своё фото с таким же именем.

## Где менять тексты

Все тексты на всех трёх языках лежат в одном файле: `lib/content.ts`. Каждый язык — отдельный блок (`et`, `en`, `ru`) с одинаковой структурой, так что переводы легко редактировать или уточнять.

## Что нового: FAQ, отзывы, Telegram, карта

- **Кнопки действия** («Забронировать», «Получить предложение» и т.д.) теперь ведут прямо в Telegram — `https://t.me/Andrei_Time_to_Surf`. Поменять юзернейм можно в `lib/constants.ts` (`TELEGRAM_USERNAME`).
- **Блок FAQ + Отзывы** — перед финальным CTA. Слева — аккордеон с вопросами (тексты в `lib/content.ts` → `faq.items`), справа — карусель отзывов (по 3 штуки, самые новые сверху, стрелки листают по 3). Отзывы сейчас — статичный список-заглушка в `lib/content.ts` → `reviews.items`, оформлены как уже одобренные модератором.
- **Кнопка «Оставить отзыв»** открывает форму (имя, компания, оценка звёздами, текст). Сейчас форма просто показывает «Спасибо» — **бэкенда пока нет**. Когда будете подключать Supabase (или любую другую БД): в `components/ReviewModal.tsx` в функции `handleSubmit` нужно отправить данные на сервер вместо простого `setSubmitted(true)`, а в `lib/content.ts` → `reviews.items` — подтягивать одобренные отзывы из базы вместо статичного массива.
- **Футер** — переработан по структуре с примера: бренд + соцсети (Facebook, Instagram, Telegram — реальные ссылки уже стоят), колонка с навигацией, колонка с контактами (телефон, email, Telegram).
- **Карта** — компактный блок в футере: встроенная карта Google Maps (без API-ключа, просто `output=embed`) + адрес / как добраться / парковка + кнопка «Открыть в Google Maps». Координаты/запрос для карты — в `lib/constants.ts` → `MAPS_QUERY`.
- **Контакты для проверки**: телефон и Telegram — из вашего профиля; email — `info@timetosurf.ee` (как на прошлом сайте), Facebook/Instagram — `timetosurf.ee`. Если что-то из этого нужно поменять — всё в `lib/constants.ts`.

## Где менять телефон / цену

Телефон, email, Telegram и ссылки на соцсети теперь в одном месте — `lib/constants.ts`. Поменяй значения там, и они обновятся везде на сайте автоматически (шапка, hero, финальный блок, футер).

Цена `€350` указана напрямую в `components/Pricing.tsx`.

## Структура проекта

```
app/            — страницы, layout, глобальные стили
components/     — все секции сайта (Hero, Gallery, Pricing и т.д.)
lib/            — тексты (content.ts), языковой контекст, генератор галереи
public/         — фото, видео, hero.jpg
```
