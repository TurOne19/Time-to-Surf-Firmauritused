# Time to Surf media slots

Сайт уже готов под реальные медиа без админки.

## Hero и ключевые секции

- `assets/photos/hero-corporate-sea.webp` - широкий hero кадр команды у моря.
- `assets/photos/safety-briefing-equipment.webp` - инструктаж, оборудование, инструктор.
- `assets/photos/final-cta-sea-background.webp` - море/команда для финального CTA.

## Галерея

Положить 64 фото:

- `assets/photos/gallery-01.webp`
- `assets/photos/gallery-02.webp`
- ...
- `assets/photos/gallery-64.webp`

Положить 4 видео:

- `assets/videos/video-01.mp4`
- `assets/videos/video-02.mp4`
- `assets/videos/video-03.mp4`
- `assets/videos/video-04.mp4`

Лайтбокс уже подключен. Если файла нет, откроется аккуратная заглушка с именем слота.
